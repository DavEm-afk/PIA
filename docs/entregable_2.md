# Entregable 2 — Adquisición de procesos y archivos

Durante este entregable se completó la **Tarea 1**, enfocada en la adquisición de datos del sistema.  
Se desarrolló un script en PowerShell, ejecutado desde Python, que obtiene información sobre **procesos activos** y **archivos relevantes**, generando salidas en formato JSON dentro de `/src/acquisition/raw/`.

También se añadieron las carpetas de evidencias (`/examples/`) y registros (`/logs/`), con archivos `.jsonl` que documentan la ejecución del proceso.

Como siguiente paso, se propone una **Tarea 3** centrada en el análisis de **conexiones de red activas**, para complementar los datos obtenidos y fortalecer la etapa de detección de anomalías.
